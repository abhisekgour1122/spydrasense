NETRA™ N & T — Device Details & Requirements

Sensor device requirements for the Neuro-Leadership (Neuroplastic & Techno-Sapient) assessment

1Core requirement

The assessment reads two things at once, so the device (single unit or a paired set) must provide BOTH:

  • EEG (brain activity) → drives T (Focus, Brain Fatigue, Mental Pressure)
  • PPG or ECG (heart rhythm → HRV) → drives N (SDNN, RMSSD, and the HRV-based measures)

An EEG-only device is not sufficient — without a heart/HRV sensor, the N index cannot be produced. A single device with both (or an EEG device paired with a dedicated HRV sensor) is required.

3The eleven readings the device must support

These feed the two indices (either output directly by the device pipeline, or derived by us from the raw EEG bands + HRV):

For N (adaptive)For T (cognitive-readiness)
SDNN, RMSSD, Stress Resistance, Brain Balance, Fatigue, LF/HF · HRFocus, Brain Fatigue, Mental Pressure, Relaxation

Emotional State is captured but held (unweighted) until polarity is confirmed.

5Recommendation

  • If a consumer device is preferred — Muse 2 / Muse S is the strongest single-unit choice: it provides EEG + PPG together, at 256 Hz, with a public SDK.
  • If HealSphere's own device is used — best path: it already outputs all eleven readings. We would need its SDK/API documentation.
  • If a research-grade EEG is preferred (e.g. Emotiv/BrainBit) — pair it with a Polar H10 chest strap for clean HRV, since those EEG units do not include a heart sensor.

Bottom line: the one non-negotiable is EEG + HRV together. Muse 2/S delivers both in a single headband and is our default recommendation unless HealSphere's device is used.
